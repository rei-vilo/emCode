# emCode {{ version }} release notes

​emCode is the continuation of embedXcode.

emCode relies on Visual Studio Code and runs on Linux, including Windows Linux Sub-system (WSL).

## Scope

Release 14.2.3 updates the ESP32 boards.

Release 14.2.3 updates the Raspberry Pi Pico RP2040 boards.

Release 14.2.0 adds the [core target](../Develop/Other.md).

Release 14.1.6 is the initial public release of emCode.
