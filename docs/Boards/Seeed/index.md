# Manage the Seeed boards

![](img/Logo-064-Seeed.png) The Seeed boards include the Grove Beginner for Arduino, the Xiao, the Wio Terminal and the SenseCAP Indicator.

The installation is performed with Arduino-CLI.

The Seeed boards includes different platforms.

+ [Grove Beginner Kit for Arduino](./Grove_AVR.md) :octicons-link-16:
+ [Xiao ESP32-C3 and -S3](./Xiao_ESP32.md) :octicons-link-16:
+ [Xiao nRF52840](./Xiao_nRF.md) :octicons-link-16:
+ [Xiao nRF52840 Mbed-OS](./Xiao_mbed.md) :octicons-link-16:
+ [Xiao RP2040](./Xiao_RP2040.md) :octicons-link-16:
+ [Xiao M0 and Wio Terminal](./Xiao_M0.md) :octicons-link-16:
+ [SenseCAP Indicator](SenseCAP.md) :octicons-link-16:
