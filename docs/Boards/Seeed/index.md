# Manage the Seeed boards

![](img/Logo-064-Seeed.png) The Seeed boards include the Grove Beginner for Arduino, the Xiao, the Wio Terminal and the SenseCAP Indicator.

The installation is performed with Arduino-CLI.

The Seeed boards includes different platforms.

+ [Grove Beginner Kit for Arduino](./Grove_AVR.md)
+ [SenseCAP Indicator](SenseCAP.md)
+ [Xiao ESP32-C3 and -S3](./Xiao_ESP32.md)
+ [Xiao M0 and Wio Terminal](./Xiao_M0.md)
+ [Xiao nRF52840](./Xiao_nRF.md)
+ [Xiao nRF52840 Mbed-OS](./Xiao_mbed.md)
+ [Xiao RP2040](./Xiao_RP2040.md)
+ [Xiao Expansion board](./Xiao_Expansion.md)

## Check the tests

The test protocol includes building and linking, uploading and running a sketch on the boards using those versions of the IDEs and plug-ins. Boards packages are versioned but not dated.

| | Platform | Package | Comment
---- | ---- |---- | ----

| | nRF52 | 1.1.7 | For Xiao nRF52840 board
