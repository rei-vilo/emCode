![](img/Logo-064-Launchpad.png) The LaunchPad CC3200 WiFi requires a specific hardware configuration.

+ Unplug the LaunchPad CC3200 WiFi board.

+ Remove the wire from `JTAG J8` (emulator side) to `SOP 2` (CC3200) side.

+ Place the `TCK J8` jumper.

+ Place the `SOP 2` jumper.

<center>![](img/408-01-420.png)</center>
<center><i>Debugging requires `TCK` and `SOP 2` jumpers placed on</i></center>

+ Plug the LaunchPad CC3200 WiFi board.
