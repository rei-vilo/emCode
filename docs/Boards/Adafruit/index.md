# Manage the Adafruit platform

![](img/Logo-064-Adafruit.png) The Adafruit platform includes the Flora, Gemma, Trinket, Trinket Pro and Feather boards.

The installation is performed with Arduino-CLI.

The Adafruit Feather boards includes different platforms.

+ [Feather ESP8266](./ESP8266.md) :octicons-link-16:
+ [Feather ESP32](./ESP32.md) :octicons-link-16:
+ [Feather nRF52832 and nRF52840](./nRF52.md) :octicons-link-16:
+ [Feather M0 and M4](./Cortex-M.md) :octicons-link-16:
+ [Feather RP2040](./RP2040.md) :octicons-link-16:
+ [Trinket and Trinket Pro AVR](./AVR.md) :octicons-link-16:

## Check the tests
